CSS

  • Enter your CSS notes here
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can change appearance of all element by setting their style.
  • Each element can receive style information from different sources - directly inside element, or from specific css file
  • Internal or inline styles are possible, but external css is more flexible.
